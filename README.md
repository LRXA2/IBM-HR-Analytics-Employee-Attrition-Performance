# Employee Attrition Prediction

**Project Goal:**
Predict employee attrition using only information available to employers at the time of hiring, based on the [IBM HR Analytics Employee Attrition & Performance](https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset) dataset from Kaggle.

**Note:** The raw dataset is not included in this repository. Please download it from Kaggle and place it in a `data/` folder if needed.
